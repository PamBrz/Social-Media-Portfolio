Explain Buzzwords - HTML, CSS, JavaScript, JSON

HTML: HTML is the standard markup language used to create and design the structure of web pages. It provides a way to organize and format content such as text, images, links, and other elements on a webpage.

HTML uses a system of tags to define different elements on a webpage.

CSS: CSS is a style sheet language that controls the presentation and layout of HTML documents. It enables the separation of content and design, allowing developers to define the visual appearance of a webpage. CSS can control aspects like colors, fonts, spacing, and positioning.

JavaScript: JavaScript is a programming language that enables interactivity and dynamic behavior on websites. It's commonly used to create responsive and engaging user interfaces. JavaScript can be embedded directly into HTML documents or included in separate files.

JSON: JSON is a lightweight data interchange format that is easy for humans to read and write, and easy for machines to generate. It is commonly used for transmitting data between a server and a web application, serving as a data format for APIs (Application Programming Interfaces).
