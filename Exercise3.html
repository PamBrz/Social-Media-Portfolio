Instagram Analysis: API Design and Request Types


Instagram API = Allows developers access to Instagrams data

HTTP Methods

POST: upload media, add comments

Input: User-Id, Media content

Output: information about uploaded media


PUT: Edit captions on postings and comments

Input: Media-Id

Output: information about added comments


DELETE: Delete comments/posts/profiles

Input: Media-Id, Comment-Id

Output: Confirmation of the deleted elements


GET: User profiles, retrieve media, search media by hashtag, get comments on a post, get followers/following list, retrieve story

Input: User-Id

Output: Username, profile picture, follower count
